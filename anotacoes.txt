PHP - iMasters

# Passo a Passo:
- Criar pasta do projeto e acessar pelo prompt.
- Iniciar projeto com composer.
    composer init
- Adicionar autoload.
    composer dumpautoload
- Adicionar o projeto no GitHub.
- Adicionar o ".gitignore".
- Iniciar o servidor PHP.
    php -S localhost:8000
